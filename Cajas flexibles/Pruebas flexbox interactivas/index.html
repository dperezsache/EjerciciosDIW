Prueba flex grow y flex shrink

El próposito de esta prueba es mostrar el funcionamiento de flex-grow y flex-shrink, mediante la inserción y borrado de contenedores.

  • flex-grow: las cajas grises tienen 3 de grow, las blancas 1, sirve para indicar la cantidad de espacio disponible que ocuparán los contenedores.
  • flex-shrink: las cajas grises tienen 3 de shrink, las blancas 1, sirve para indicar la cantidad de espacio que se le quitarán a los contenedores en caso que sea necesario.

Prueba flex direction (row)

El próposito de esta prueba es mostrar los diferentes tipos de dirección que pueden tomar los contenedores con flex.

  1. row: los contenedores se alinean a la izquierda.
  2. row-reverse: los contenedores se alinean a la derecha, y los contenedores se quedan en orden inverso.
  3. column: los contenedores se alinean de arriba a abajo.
  4. column-reverse: los contenedores se alinean de arriba a abajo, y los contenedores se quedan en orden inverso

align-items (stretch)

El próposito de esta prueba es probar como los contenedores se pueden alinear sobre el eje de cruz (verticalmente) con align-items.

  1. stretch: Los contenedores se estiran para ocupar todo el espacio vertical.
  2. flex-start: Los contenedores se pegan al top del contenedor padre.
  3. flex-end: Los contenedores se pegan al bottom del contenedor padre.
  4. center: Los contenedores se ponen al centro del contenedor padre.
  5. baseline: Los contenedores se alinean sobre la línea de la fuente de texto.

gap

El próposito de esta prueba es probar el funcion del atributo gap, el cúal sirve para añadir espacio entre contenedores flex.

  1. gap: Propiedad shorthand para cambiar tanto 'row-gap' como 'column-gap'.
  2. row-gap: Espaciado vertical entre contenedores.
  3. column-gap: Espaciado horizontal entre contenedores.
